# Speed to Lead - Dashboard

React dashboard for visualizing speed-to-lead metrics.

## Quick Start

1. Install dependencies: `npm install`
2. Start development server: `npm start`
3. Open http://localhost:3000

## Configuration

The dashboard connects to the backend API. By default, it expects the API at `http://localhost:3000`.

To change the API URL, set the `REACT_APP_API_URL` environment variable or modify `src/api.js`.

## Features

- Real-time dashboard with key metrics
- Overall statistics and percentile analysis
- Setter performance reports
- Interactive charts and tables
